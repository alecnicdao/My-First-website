How to Compose URL

So first off you have to use a certain protocol. In our case we are using the http:// protocol then the server's name.

  1. Protocol, which is http://
  2. Server or domain name, which is icarus.cs.weber.edu~username/
  3. After the slash after username you type in where you want it to go
  4. For example it is suppose to look like this http://icarus.cs.weber.edu/~an57205/distractions.html
