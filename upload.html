How to Upload to Icarus Server

So first off you will need to have a server. So we can store your files on a machine that is specifically a web server. What web servers do is they store files and wait for requests, and when a request comes they return the file. A request can come when you type in a specific URL. And when you type in a URL it returns that file. So to get server space here at WSU the CS department has provided everyone in a CS class their own server space and it's on the server called "Icarus".

Steps on how to make an account

  1. www.weber.edu/cs (should take you to the main page of the CS)
  2. Click on technical support
  3. Scroll down until you see "Default Login Help" (click it)
  4. Scroll down until you see Default Login Tool (click it)
  5. Enter your first and last name and including your W Number (click submit)
  6. The one we are going to use is the green one. (Icarus/Linux)

How to move your files

So then we need to be able to move our files. What we need is a File Transer Protocol Client (FTP). For example, you can download Cyberduck, which is a FTP client you can download for free. It is great, because it supports many platforms (windows, mac, linux). There are many out there so feel free to use the one you are most comfortable with.

Once you have downloaded Cyberduck

  1. Click on Open Connection
  2. Change protocol to SFTP (SSH File Transfer Protocol) Port 22
  3. Type in server information (icarus.cs.weber.edu)
  4. Type in username and password
  5. Click connect

How to upload your files/folders

  1. Go into public_html folder
  2. After that you can drag and dropyour files into that file once you have clicked on the public_html folder
  3. Another way is you can click the upload button
  4. It is also the same if you wanted to drap and drop an entire folder
